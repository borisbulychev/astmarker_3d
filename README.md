# Задача – работа с 3D-представлением

## Описание

**Целью** работы является создание приложения, переводящего 2D-развертку (PNG + TIFF с координатами X,Y,Z) обратно в 3D-модель. Топология поверхности – диск, незамкнутая.

## Подзадачи

- Интерактивный выбор области (ROI)
- Построение 3D mesh
- Визуализация через libigl
- Экспорт в OBJ
- Контроль детализации
